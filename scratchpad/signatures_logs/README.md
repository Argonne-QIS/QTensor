# Log of operations performed


## signatures only

files: `log_localhost_signatures_N` where `N` is the size of task

## signatures with bucket info


files: `log_with_buckets_N` where `N` is the size of task


Signature format:
`list, list -> list`

for example:

```T_ijk * T_klj = [1,2,3], [3,4,2] -> [1,2,3,4]```
